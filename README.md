# 🚀 کنترل موس با دست 


**یک برنامه پایتونی جهت سرگرمی برای کنترل موس سیستم با دست.**

___


**✨ ویژگی‌های کلیدی**

1- ژست اسکرول: انگشت اشاره و وسط باز (V شکل)

2- ژست درگ: مشت بسته (همه انگشت‌ها جمع)

3- کلیک راست (وسط + شست)

4- دابل کلیک: سبابه + شست (1.5 برابر آستانه)

5- کلیک چپ (سبابه + شست) - آستانه معمولی

---

# 🛠️ نحوه راه‌اندازی و اجرا

## ۱. پیش‌نیازها
برای اجرای این ابزار  باید `python 3` روی سیستم شما نصب باشد.

**همچنین این پکیج هارا باید داشته باشید ( اگر ندارید این کد را کپی و در ترمینال پیست کنید)**

```bash
pip install opencv-python mediapipe pyautogui numpy 
```

## ۲. اجرا روی سیستم میزبان
`ترمینال` یا `CMD` خود را در پوشه پروژه باز کرده و دستور زیر را تایپ کنید:

```bash
python mouse_hand.py

```
یا روی پروژه دابل کلیک کنید!
